This is the API

registerBuilding

플랫폼에 집 주인이 추가되고, 집 주인이 건물을 등록하는 기능이 생긴다면 건물을 직접 등록하는 api도 필요할 것

URL

https://virtserver.swaggerhub.com/QHTNSDLAKFXL/Project/1.0.0/buildings/{buildingId}

HTTP Method

POST

Response Type

Object

Parameters

  • Path: buildingId Integer

    빌딩의 numeric ID

registerRoomsInBuilding

집 주인은 본인의 건물을 먼저 추가하고 해당하는 건물에 들어있는 호실 정보를 한번에 입력할 수 있다.

URL

https://virtserver.swaggerhub.com/QHTNSDLAKFXL/Project/1.0.0/building/{buildingId}/room

HTTP Method

POST

Response Type

Parameters

  • Body: body List(Room)

    호실 정보를 추가하기 위한 roomsList

  • Path: buildingId Integer

    빌딩의 numeric ID
